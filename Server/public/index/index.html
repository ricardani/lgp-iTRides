Uma plataforma de car-sharing da iTGrow

Conheça a Nossa Aplicação

iTRides é uma aplicação de partilha de boleias exclusiva para os contribuidores das empresas iTGrow e Critical Software.



Objetivos

Consciência Ambiental

Ajuda a conservar o meio ambiente, diminuindo o número de carros em circulação.

Redução de Custos

Se és condutor, divide as tuas despesas partilhando os teus lugares vagos.
Se és passageiro, viaja de forma mais económica.

Integração

Uma maneira fácil de conheceres melhor os teus colegas de trabalho.





Funcionalidades

Boleias Pré-definidas
Para viagens diárias entre o trabalho e a sua casa
Boleias Personalizadas
Para viagens menos frequentes, com data fixa
Boleias Ocasionais
Para deslocações esporádicas para fora da cidade
Sistema de Pontuação
Para uma fácil avaliação dos utilizadores
Interface simples e fluida
Para uma utilização intuitiva
Integração com o calendário do android
Para uma melhor gestão do tempo

Faz Download da Plataforma

Ajuda o ambiente, poupa nas tuas viagens e integra-te facilmente com os teus colegas de trabalho. Aproveita a nossa aplicação, disponível para Android 4.2 e superior.
